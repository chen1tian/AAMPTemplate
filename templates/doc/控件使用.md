# 控件使用

此项目集成了`Semi.Avalonia`以及`Ursa`两个控件库，可以直接使用。

## Semi.Avalonia 使用

说明文档：https://github.com/irihitech/Semi.Avalonia/blob/main/README_CN.md


## Ursa 使用

说明文档：https://github.com/irihitech/Ursa.Avalonia/blob/main/README.zh-CN.md

```
<Window
    ...
    xmlns:u="https://irihi.tech/ursa"
    ...>
    <StackPanel Margin="20">
        <u:TagInput />
    </StackPanel>
</Window>
```
